# Swipe Car Game

## 📌 프로젝트 소개
마우스 스와이프 입력을 이용해 자동차를 이동시키는 간단한 물리 기반 2D 게임

## 🎮 주요 기능
- 마우스 스와이프 기반 이동 입력
- 스와이프 길이에 따른 속도 변화
- 감속(관성) 효과 적용
- 목표 지점까지 거리 UI 표시

## 🛠 사용 기술
- Unity :contentReference[oaicite:0]{index=0}
- C#

## 🧠 구현 핵심 로직
- 스와이프 시작/끝 좌표 차이로 이동 방향 및 힘 계산
- 속도 값을 기반으로 Transform 이동 처리
- 일정 비율로 속도 감소하여 자연스러운 정지 구현

## 📁 구조 특징
- 입력 처리와 이동 로직 분리
- UI는 거리 계산 결과만 표시
